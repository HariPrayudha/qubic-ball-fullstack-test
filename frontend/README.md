# Frontend — Next.js 16

User & admin UI for the Qubic Support Ticket System.
Stack: Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS 4, shadcn/ui,
TanStack Query, React Hook Form + Zod.

See the [root README](../README.md) for full setup, environment variables, and architecture.

```bash
cp .env.example .env.local
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
```
